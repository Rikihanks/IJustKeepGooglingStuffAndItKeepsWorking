ZEUS CODE GENERATOR

Zeus is born to fix the repetitive boilerplate code that we are used to see in DAO based applications. It uses a simple interface that will be fed with the entities data of your project and generate the services, interfaces and DTOs based on the entities that you have provided. You can then download and import the code to your favourite IDE saving lots of time.
